# NAND to Tetris

My solutions for the [NAND to Tetris](http://www.nand2tetris.org/) course.

## Structure
- `01-06`: Hardware (HDL: gates, ALU, CPU, etc.)
- `07-12`: Software (VM, compiler, OS, Tetris)(upcoming)

## Setup
1. Download tools from [nand2tetris.org/software](http://www.nand2tetris.org/software)
2. Run with HardwareSimulator, Assembler, or JackCompiler
